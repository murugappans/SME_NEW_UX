Minimum System Requirements

 

  Web Server

  • .Net Framework 2.0 {Included in the installation disk}
  • Windows XP Professional*  or above / Windows server 2003 or above
  • IIS 5.0 or above
  • 512 megs RAM
  • Hard drive Space: 100 megs
  • Database: SQL EXPRESS 2005 {Included in the installation disk}

*Note: Windows XP / Vista or Windows 7 Home editions do not include IIS, therefore are not supported.

 

 

  Client PC

  • IE 6.0+
