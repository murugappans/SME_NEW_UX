What type of CPF contribution sectors are supported?

Smepayroll currently supports only Private sector for calculation of CPF Contribution



The selection for the sector is done under the employee profile - Personal Info tab, under the same
tab the Employer CSN is also selected.



Futher information can be found on CPF BOARD website - www.cpf.gov.sg
